# Dashboard Administrativo

## Descripción

Dashboard administrativo desarrollado con HTML5 y CSS3 utilizando CSS Grid y Flexbox.

## Tecnologías usadas

- HTML5
- CSS3
- CSS Grid
- Flexbox

## Características

- Diseño responsive
- Sidebar adaptable
- Tarjetas interactivas
- Tabla de datos
- Efectos hover
- Accesibilidad básica

## Diseño

El dashboard fue diseñado siguiendo tendencias modernas inspiradas en dashboards administrativos profesionales.
